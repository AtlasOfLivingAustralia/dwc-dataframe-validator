## Example usage for darwin core archives

```python
import jsonpickle
from dwca.read import DwCAReader
from dwc_validator.validate_dwca import validate_archive

with DwCAReader("/tmp/my-dwca.zip") as dwca:
    # Validate a data frame
    validation_report = validate_archive(dwca)
    # View report in JSON
    print(jsonpickle.encode(validation_report, unpicklable=False, indent=2))
```
